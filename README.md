# vite_intro
Objective
Ship a feature slice that proves routing, state, and effect.

Requirements
At least two routes and a shared layout wrapper.
One stateful component that updates its view from user input.
One effect that fetches data and renders loading and error states.
One small transform (map, filter, or reduce) applied before rendering.
README shows install, start, and where to view the feature.
